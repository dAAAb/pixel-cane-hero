# 像素拐杖侠

一个极简像素风格的横向卷轴游戏，主角是穿绿色西装、头发有点秃、手拿拐杖的老先生。

## 游戏特点

- 横向卷轴像素风格游戏，类似超级马里奥兄弟
- 主角：穿绿色西装的老先生，拄着拐杖走路
- 拐杖作为武器，可以攻击敌人
- 特殊机制：拐杖有时会断裂并射出，可以击穿所有敌人

## 游戏控制

- **左右方向键**：移动
- **上方向键**：跳跃
- **空格键**：挥动拐杖攻击
- **游戏结束后按R**：重新开始游戏

## 安装与运行

1. 安装依赖：
```
pip install -r requirements.txt
```

2. 运行游戏：
```
python main.py
```

## 游戏机制

- 老先生可以用拐杖攻击敌人
- 有1/3的几率拐杖会断裂并射出，可以穿透所有敌人
- 拐杖断裂后需要等待一段时间才能恢复
- 碰到敌人游戏结束
- 击败敌人可以获得分数
